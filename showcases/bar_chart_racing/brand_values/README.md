# Brand Values Bar Chart Race

Animated bar chart showing the top company brand values over time. Bars slide and re-rank as values change year by year, with a dynamic axis that rescales automatically and a large year counter in the background.

## Run

```bash
python showcases/bar_chart_racing/brand_values/bar_chart_race.py
```

## Config

Edit `config.toml` to change the data source, number of top bars, color palette, fonts, margins, and export settings.
