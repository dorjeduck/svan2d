# Periodic Table Discovery Timeline

Elements of the periodic table appear one by one in their grid positions, ordered by discovery year. A large background year label updates as each element appears. Ancient elements (copper, gold, iron) appear first, progressing through to modern synthetic elements.

## Run

```bash
python showcases/periodic_table/discovery_timeline/main.py
```

## Config

Edit `config.toml` to change colors, fonts, animation timing, and export settings.
