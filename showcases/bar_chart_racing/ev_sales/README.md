# EV Sales Bar Chart Race

Animated bar chart showing global electric vehicle sales by country from 2010 to 2023 (IEA data). Countries slide and re-rank as sales volumes change, colored by geographic region with a rescaling axis and year display.

## Run

```bash
python showcases/bar_chart_racing/ev_sales/bar_chart_race.py
```

## Config

Edit `config.toml` to change the data source, number of top bars, region color palette, fonts, and export settings.
